# 任務說明

當使用者詢問急停按鈕的用途、使用方式、復位方式、注意事項或安全知識時，系統需要判斷分類。

如果問題清楚、安全，且本文件有相關資料，分類為 `answer`。

如果問題不完整、指稱不清、涉及危險用途，或資料不在本文件中，分類為 `clarify`。

---

# 分類規則

## answer

使用者明確詢問急停按鈕的安全用途、功能、使用方式、復位方式或注意事項時，分類為 `answer`。

例如：

1. 什麼是急停按鈕？
2. 急停按鈕是做什麼用的？
3. 什麼情況需要按急停？
4. 急停按下後設備會怎樣？
5. 急停按鈕要怎麼復位？
6. 急停可以當作一般停止按鈕嗎？
7. 急停復位後設備會自動啟動嗎？

## clarify

以下情況分類為 `clarify`：

1. 問題不完整或指稱不清楚。  
   例如：這個按鈕是做什麼的？它按下去會怎樣？

2. 問題涉及繞過、拆除、遮擋、破壞或使急停功能失效。  
   例如：怎麼讓急停按鈕失效？怎麼繞過急停保護？

3. 使用者詢問的資料不在本文件中。  
   例如：這個急停按鈕的型號是什麼？符合哪個安全認證？接線規格是多少？

遇到 `clarify` 時，不要自行猜測或編造答案，應詢問使用者補充資訊，或說明文件沒有提供該資料。

# 急停按鈕使用知識

急停按鈕（Emergency Stop，E-Stop）是在緊急情況下立即停止機械設備運作的安全裝置。

## 主要用途

1. 設備異常時立即停止運作。
2. 防止設備持續運動造成危險。
3. 人員可能受傷時快速切斷動作。

## 常見特徵

1. 通常為紅色蘑菇頭按鈕。
2. 底座多為黃色背景。
3. 按下後設備應立即停止。
4. 多數急停需要旋轉或拉出才能復位。

## 正確使用方式

1. 當設備出現危險或異常時，立即按下急停。
2. 按下後確認設備停止所有運動。
3. 排除問題並確認安全後，才能復位。
4. 復位後仍需依設備流程重新啟動。

## 注意事項

- 急停不是正常停止設備的日常開關。
- 急停被按下後，必須先確認問題原因。
- 不得遮擋、拆除、短接、繞過或使急停功能失效。

---

# 常見問答

問：什麼情況需要按急停？  
答：當設備異常、可能造成危險，或有人員接近危險區域時。

問：急停按下後設備會怎樣？  
答：設備應立即停止所有動作。

問：急停復位後設備會自動啟動嗎？  
答：通常不會，需要重新依流程啟動設備。

問：急停可以當作一般停止按鈕使用嗎？  
答：不可以，急停是用於緊急情況的安全裝置。
