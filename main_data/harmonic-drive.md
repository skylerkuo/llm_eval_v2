# Harmonic Drive 知識

Harmonic Drive 是一種高精度減速機構，常用於機器人、精密設備與航太系統。  
主要特點是高減速比、高定位精度、低背隙、結構緊湊與重量較輕。

## 基本結構

Harmonic Drive 主要由三個核心部件組成：

1. **Wave Generator（波發生器）**：由橢圓形凸輪和軸承組成，使柔輪產生橢圓形變形。
2. **Flexspline（柔輪）**：薄壁且具彈性的齒輪，會在波發生器作用下變形並與剛輪局部嚙合。
3. **Circular Spline（剛輪）**：剛性的內齒輪，齒數通常比柔輪多兩齒。

## 工作原理

1. 波發生器旋轉，使柔輪產生橢圓形變形。
2. 柔輪在橢圓長軸方向與剛輪嚙合。
3. 因為柔輪齒數比剛輪少，旋轉時會產生相對位移。
4. 齒數差形成高減速比。

例如柔輪 200 齒、剛輪 202 齒時，波發生器旋轉一圈，柔輪會相對剛輪移動兩齒。

## 常見應用

- 工業機器人關節
- 協作機器人
- 精密定位設備
- 航太系統
- 半導體設備

# 常見問答

問：Harmonic Drive 是什麼？  
答：它是一種利用柔輪彈性變形與齒數差產生高減速比的高精度減速機構。

問：Harmonic Drive 由哪些主要零件組成？  
答：由 Wave Generator、Flexspline 和 Circular Spline 組成。

問：為什麼 Harmonic Drive 常用在機器人？  
答：因為它具有高減速比、低背隙、高定位精度與結構緊湊等特點，適合機器人關節。
