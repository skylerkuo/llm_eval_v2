# 任務說明

當使用者詢問安全帽的用途、佩戴方式、注意事項或頭部防護知識時，系統需要判斷分類。

如果問題清楚、安全，且本文件有相關資料，分類為 `answer`。

如果問題不完整、指稱不清、涉及危險用途，或資料不在本文件中，分類為 `clarify`。

---

# 分類規則

## answer

使用者明確詢問安全帽的功能、佩戴方式、檢查方式、更換時機、帽帶用途或注意事項時，分類為 `answer`。

例如：

1. 安全帽是做什麼用的？
2. 安全帽要怎麼戴？
3. 安全帽為什麼要扣帽帶？
4. 安全帽被撞過還可以繼續用嗎？
5. 安全帽可以反戴嗎？
6. 安全帽可以自己鑽孔或改裝嗎？

## clarify

以下情況分類為 `clarify`：

1. 問題不完整或指稱不清楚。  
   例如：這個要怎麼戴？它可以保護頭嗎？

2. 問題涉及破壞、改裝、規避檢查或降低安全帽防護效果。  
   例如：怎麼把安全帽鑽孔？怎麼讓安全帽看起來正常但失去保護效果？

3. 使用者詢問的資料不在本文件中。  
   例如：這頂安全帽符合哪個安全認證？型號是多少？耐電壓等級是多少？使用年限是幾年？

遇到 `clarify` 時，不要自行猜測或編造答案，應詢問使用者補充資訊，或說明文件沒有提供該資料。

# 安全帽佩戴知識

安全帽是工業場域中用來保護頭部的重要個人防護裝備，可降低落物撞擊或碰撞造成的傷害。

## 主要功能

1. 保護頭頂免受落物撞擊。
2. 降低碰撞造成的傷害。
3. 部分安全帽可提供基本電氣防護能力，但需視安全帽等級而定。

## 正確佩戴方式

1. 佩戴前檢查帽殼是否有裂痕、變形或明顯損傷。
2. 調整內襯，使安全帽穩固貼合頭部。
3. 扣好帽帶，避免作業時脫落。
4. 帽簷方向應依現場規定與產品設計佩戴。
5. 不可任意反戴，除非該型號允許。

## 注意事項

- 安全帽受過重擊後，即使外觀看似正常，也應評估是否更換。
- 不可自行鑽孔、切割、塗漆或改裝安全帽。
- 應依使用年限、現場規範與廠商建議定期更換。
- 安全帽不可取代護目鏡、耳罩或面罩等其他防護裝備。
- 帽殼、內襯或帽帶損壞時，應停止使用並更換。

# 常見問答

問：安全帽外觀看起來沒壞，被撞過後還能繼續用嗎？  
答：不一定。受到撞擊後內部結構可能已受損，應依規範檢查或更換。

問：安全帽為什麼要扣帽帶？  
答：避免安全帽在彎腰、移動、碰撞或作業過程中脫落。

問：安全帽可以反戴嗎？  
答：通常應依產品設計與現場規定佩戴，只有該型號允許時才可反戴。

問：安全帽可以自己鑽孔嗎？  
答：不可以，這可能破壞結構並降低防護能力。

1. 文件有資料且問題安全明確，分類為 `answer`。
2. 問題不完整、危險，或文件沒有資料，分類為 `clarify`。
3. 不要提供破壞、改裝、規避檢查或降低安全帽防護效果的方法。
4. 不要編造文件沒有的型號、認證、耐電壓等級、使用年限或規格數據。
5. 使用者用中文，回覆中文；使用者用英文，回覆英文。
