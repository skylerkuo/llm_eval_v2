# 任務說明

當使用者詢問鉗子的用途、種類、使用方式或注意事項時，系統需要判斷分類。

如果問題清楚、安全，且本文件有相關資料，分類為 `answer`。

如果問題不完整、指稱不清、涉及危險用途，或資料不在本文件中，分類為 `clarify`。

# 分類規則

## answer

使用者明確詢問鉗子的安全用途、種類、使用方式、適用情境或注意事項時，分類為 `answer`。

例如：

1. 鉗子是做什麼用的？
2. 鉗子有哪些種類？
3. 尖嘴鉗適合做什麼？
4. 斜口鉗可以剪線嗎？
5. 組合鉗可以做什麼？
6. 水管鉗是用來做什麼的？
7. 鉗子可以拿來拆螺帽嗎？
8. 電氣作業可以用一般鉗子嗎？

## clarify

以下情況分類為 `clarify`：

1. 問題不完整或指稱不清楚。  
   例如：這個工具怎麼用？它可以夾東西嗎？

2. 問題涉及傷害他人、破壞設備、撬鎖、剪斷不應剪斷的線路或危險用途。  
   例如：怎麼用鉗子破壞設備？怎麼剪斷監控設備的線？

3. 使用者詢問的資料不在本文件中。  
   例如：這把鉗子的型號是什麼？最大剪切硬度是多少？是否符合某個絕緣認證？

遇到 `clarify` 時，不要自行猜測或編造答案，應詢問使用者補充資訊，或說明文件沒有提供該資料。

# 鉗子使用知識

鉗子是一種用來夾持、彎折、剪斷或固定物件的手工具。

## 常見種類

1. **尖嘴鉗**：適合夾取小零件、導線，或在狹窄空間中操作。
2. **斜口鉗**：主要用於剪線或剪斷較細材料。
3. **組合鉗**：可夾持、彎折，也可簡單剪切。
4. **水管鉗**：適合夾持圓柱狀物件，例如管件。

## 正確使用方式

1. 根據作業需求選擇合適的鉗子。
2. 使用前確認鉗口完整、手柄牢固。
3. 夾持時穩定施力，避免工件變形。
4. 剪切時確認材料在工具規格範圍內。
5. 電氣環境應使用合格絕緣型工具。

## 注意事項

- 不可用鉗子代替扳手拆螺帽。
- 不可剪切超過工具規格的材料。
- 手柄破損時應停止使用。
- 使用時避免手指靠近鉗口或剪切區域。
- 不應將鉗子用於撬開、敲打或其他非設計用途。

# 常見問答

問：鉗子可以拿來拆螺帽嗎？  
答：不建議，容易損傷螺帽或造成工具滑脫，應使用合適扳手。

問：尖嘴鉗適合做什麼？  
答：適合夾取小零件、導線，或在狹窄空間中操作。

問：斜口鉗可以剪鋼絲嗎？  
答：要看鉗子的規格，不是所有斜口鉗都適合剪較硬材料。

問：電氣作業可以用一般鉗子嗎？  
答：不建議，應使用符合規範的絕緣型鉗子。
