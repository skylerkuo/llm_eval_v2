# 任務說明

當使用者詢問扳手的用途、種類、使用方式或注意事項時，系統需要判斷分類。

如果問題清楚、安全，且本文件有相關資料，分類為 `answer`。

如果問題不完整、指稱不清、涉及危險用途，或資料不在本文件中，分類為 `clarify`。

---

# 分類規則

## answer

使用者明確詢問扳手的安全用途、種類、使用方式、適用情境或注意事項時，分類為 `answer`。

例如：

1. 扳手是做什麼用的？
2. 扳手有哪些種類？
3. 開口扳手適合做什麼？
4. 梅花扳手有什麼優點？
5. 活動扳手怎麼使用？
6. 套筒扳手適合什麼情況？
7. 扳手可以加裝鐵管延長施力嗎？
8. 為什麼扳手會滑脫？

## clarify

以下情況分類為 `clarify`：

1. 問題不完整或指稱不清楚。  
   例如：這個工具怎麼用？它可以拆螺栓嗎？

2. 問題涉及傷害他人、破壞設備、撬鎖、敲打設備或危險用途。  
   例如：怎麼用扳手破壞設備？怎麼把扳手改造成武器？

3. 使用者詢問的資料不在本文件中。  
   例如：這把扳手的型號是什麼？最大扭力是多少？材質規格是什麼？是否符合某個認證？

遇到 `clarify` 時，不要自行猜測或編造答案，應詢問使用者補充資訊，或說明文件沒有提供該資料。

# 扳手使用知識

扳手是用來夾持並旋轉螺帽、螺栓或六角零件的手工具。

## 常見種類

1. **開口扳手**：適合快速拆裝，但包覆性較低。
2. **梅花扳手**：包覆性較好，不易滑脫。
3. **活動扳手**：開口大小可調整，適合不同尺寸。
4. **套筒扳手**：適合較深位置、空間受限或需要穩定施力的情況。

## 正確使用方式

1. 先確認螺帽、螺栓或六角零件尺寸。
2. 選用尺寸正確且種類合適的扳手。
3. 讓扳手完整貼合工件。
4. 施力方向要穩定，避免斜向硬扭。
5. 儘量以拉動方式操作，不要猛力推壓。
6. 不要任意加裝延長桿亂加力。
7. 使用活動扳手時，確認開口已固定緊。

## 注意事項

- 尺寸不合時不可勉強使用。
- 不可用扳手敲打物件。
- 不可將扳手當作撬棒使用。
- 扳手有裂痕、變形或嚴重磨損時應停止使用。
- 使用時注意手部位置，避免滑脫受傷。

# 常見問答

問：活動扳手和固定扳手哪個比較好？  
答：固定扳手通常較穩定；活動扳手較方便，但若沒有調緊較容易滑動。

問：扳手可以加裝鐵管延長施力嗎？  
答：一般不建議，容易造成工具損壞、螺栓斷裂或人員受傷。

問：為什麼扳手會滑脫？  
答：常見原因包括尺寸不合、開口未貼合、活動扳手未調緊或施力角度不正確。

問：套筒扳手適合什麼情況？  
答：適合較深位置、空間受限或需要穩定施力的螺帽與螺栓拆裝。

# 回覆原則

1. 文件有資料且問題安全明確，分類為 `answer`。
2. 問題不完整、危險，或文件沒有資料，分類為 `clarify`。
3. 不要提供傷害他人、破壞設備、撬鎖、敲打設備或違法用途的操作方法。
4. 不要編造文件沒有的型號、規格、最大扭力、材質、認證或數據。
5. 使用者用中文，回覆中文；使用者用英文，回覆英文。
