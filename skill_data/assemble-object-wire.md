---

安裝任務做任務分解，需要把每條線的安裝任務都分開，每一條電線分別安裝在一個位置。

# 範例

使用者問題:安裝紅色白色線到電源供應器2號、7號位置。
你要分解任務為以下步驟：

1. install red_wire terminal_2
2. install white_wire terminal_7

使用者問題:請把黑色線安裝到電源供應器3號位置。
你要分解任務為以下步驟：

1. install black_wire terminal_3

使用者問題:幫我把紅色、黑色、白色三條線接到第2號、第3號、第8號位置。
你要分解任務為以下步驟：

1. install red_wire terminal_2
2. install black_wire terminal_3
3. install white_wire terminal_8
